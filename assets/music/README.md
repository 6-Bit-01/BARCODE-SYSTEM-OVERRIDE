# Music Assets

This folder contains background music tracks for BARCODE: System Override.

## Required Tracks

### main-theme.mp3
- Plays during exploration/tutorial
- Should be calm but futuristic
- Suggested length: 2-3 minutes (loops seamlessly)

### combat-theme.mp3  
- Plays during enemy encounters
- Should be intense and action-oriented
- Suggested length: 1:30-2 minutes (loops seamlessly)

### rhythm-theme.mp3
- Plays during rhythm combat mode
- Should have clear beat for timing attacks
- Must be tempo-synced (120 BPM recommended)
- Suggested length: 30-60 seconds (loops seamlessly)

## Audio Format Guidelines

- **Format**: MP3 or OGG Vorbis (smaller file size)
- **Bitrate**: 128-192 kbps (good quality/size balance)
- **Sample Rate**: 44.1 kHz
- **Channels**: Stereo
- **Volume**: Normalize to -6 dB to prevent clipping

## Loop Points

All tracks should loop seamlessly:
- Start and end at the same musical phrase
- No silence at beginning or end
- Consistent tempo throughout

## File Naming Convention

Use lowercase with hyphens:
- ✅ `main-theme.mp3`
- ✅ `combat-theme.mp3` 
- ✅ `rhythm-theme.mp3`
- ❌ `MainTheme.mp3`
- ❌ `Combat Theme.mp3`
